Aide

Affichage Images, Affichage Fiches

- Appui court sur un résultat => Affichage image
- Appui long sur un résultat => Affichage Fiche

Type de Fiches

- Le type de fiche apparait sur les résultats, les images et les fiches.
  • F : Fiche publiée
  • FR : Fiche en cours de Rédaction - La fiche peut-être incomplète.
  • FP : Fiche Proposée - Très peu d'informations sont disponibles.

Accéder aux paramètres

- Pour accéder aux paramètres de cette application, il faut afficher le menu en cliquant sur la touche Menu : de votre appareil.

Filtrage des Fiches

- Il est possible sur le site DORIS de filtrer les résultats. De même, il est possible dans cette application de n'afficher que les fiches publiées.
- Par défaut, nous proposons la recherche guidée "Toutes zones". Il est possible de restreintre la recherche à l'une des 5 zones proposées sur le site.

Paramètres de Téléchargement

- Afin d'éviter de télécharger à chaque fois les mêmes données, les pages html et les images téléchargées sur le site DORIS sont stockées sur votre appareil. Cela permet de fluidifier la navigation et de limiter le volume de données téléchargées. Par contre si une fiche évolue sur le site alors que vous l'avez déjà consultée, vous ne pourrez le constater qu'après la durée paramétrée.
